Core Capabilities Built for Rigor & Speed

QR-Based Device Pairing

Opaque, server-side tokens generate discoverable QR keys. Scan to pair instantly with zero public exposure of email or user IDs, keeping sessions strictly scoped.

Direct-to-S3 Presigned Uploads

Files stream directly to cloud object storage. Generates secure presigned URLs to completely bypass backend memory limitations and optimize file transfer speeds.

In-Browser Code Editing

Collaborate in real-time on code and text formats including `.txt`, `.md`, `.html`, `.json`, and `.csv`. Features a premium editor with word wrapping, global find & replace, and file stats.

File Versioning History

Every document save compiles a new version in S3 and logs it. Maintain a history of up to 10 versions with a single-click restore option to roll back changes.

Granular Permission Control

Manage transfer policies dynamically. Revoke or enable upload, download, and edit capabilities in real-time without invalidating existing files.

Automated Garbage Collection

Built-in background workers and database TTL monitors automatically purge guest sessions in 10 minutes and delete guest files within 24 hours of upload.

Security Architecture Checklist

A system architected from the ground up for strict confidentiality and data isolation.

  • 🛡️ RS256 Signature verification: All user logins use asymmetrical RSA 2048-bit keys.
  • 🍪 HTTP-Only Strict Cookies: Session tokens are locked to HTTP-only SameSite=Strict cookies to block XSS vector theft.
  • 🗝️ Opaque QR Tokens: QR codes encode only short-lived random bytes, containing no database IDs or user information.
  • 👤 Temporary Guest Tier: Guest users run in ephemeral sandboxes with automated 10-minute session TTLs.
  • 🔍 Binary Magic-Byte Inspection: Uploaded file types are strictly inspected using python-magic binary analysis.
